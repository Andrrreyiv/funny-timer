# 🐷 Funny Timer — Таймер 5 секунд

Простой одностраничный веб-таймер с обратным отсчётом и смешным звуком.

## Функционал

- Логотип игры (заглушка, легко заменить)
- Большая красная кнопка «НАЖМИ МЕНЯ»
- Обратный отсчёт: 5 → 4 → 3 → 2 → 1
- Смешной звук (поросячий визг) по окончании
- Автоматический сброс — цикл повторяется при следующем нажатии

## Технологии

- Чистый HTML/CSS/JS — без зависимостей
- Адаптивный дизайн — работает на любом смартфоне
- Звук генерируется через Web Audio API — не нужны внешние файлы

## Как запустить

Просто откройте `index.html` в браузере. Файлы можно разместить на любом хостинге.

## Замена логотипа

В `index.html` замените блок:
```html
<div class="logo-placeholder">ЛОГОТИП ИГРЫ</div>
```
на тег `<img>` с нужным логотипом:
```html
<img src="logo.png" alt="Логотип" class="logo-img">
```

## Замена звука

Если нужен другой звук, добавьте аудиофайл и замените вызов `createPigSqueal()` в `script.js` на воспроизведение через `<audio>` элемент.
